
## 1. Difference between var, let, const
Ans..
var holo function scoped ebong etake punoray declare kora jay.let holo block scoped ebong etake punoray declare kora jay na.
const o block scoped, kintu er value poriborton kora jay na.

## 2. What is the spread operator (...)?
Ans..
Spread operator use kora hoy array ba object er element gulo expand korar jonno...

Example:
const arr = [1,2,3];
const newArr = [...arr,4]; // [1,2,3,4]

## 3. What is the difference between map(), filter(), and forEach()?
map() notun array create kore function apply kore.filter() notun array create kore specific condition er upor based.
forEach() sudhu array er element gulo loop kore, kintu notun array return kore na.....

## 5. What are template literals? 
Template literals use kore string er moddhe variable embed kora jay backticks (`) diye.

Example:
const name = "John";
`Hello ${name}`; // "Hello John"
